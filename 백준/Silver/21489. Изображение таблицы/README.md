 <p>Выведите в выходной файл символическое изображение структуры таблицы. </p>

<p>Изображение <mjx-container class="MathJax" jax="CHTML" style="font-size: 109%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D456 TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>i</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$i$</span></mjx-container>-й строки таблицы должно начинаться изображением горизонтальной линии, составленным из символов <<<code>+</code>>> (плюс) и <<<code>-</code>>> (минус).
