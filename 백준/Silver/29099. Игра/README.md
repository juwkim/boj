### 문제 설명

<p>Джейме и Серсея играют в игру. </p>

<p>У них есть отсортированный по возрастанию массив. За один ход игрок может отнять единицу от каждого числа на некотором суффиксе массива. Проигрывает игрок, после хода которого в массиве появляется два одинаковых числа, либо одно из чисел становится равным нулю.</p>

<p>Джейме ходит первым, Серсея --- второй. Кто выиграет, если оба игрока будут играть оптимально?</p>
