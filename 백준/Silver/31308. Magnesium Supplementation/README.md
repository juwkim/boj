His doctor can prescribe pills that have any integer number of picograms of magnesium from one up to a maximum. Brandon doesn't like taking pills though, so he refuses any prescription that requires him to take too many pills in a day.</p>

<p>Your task is to find all prescriptions that Brandon could be given. Note that all pills prescribed to Brandon must have the same dosage.</p>
