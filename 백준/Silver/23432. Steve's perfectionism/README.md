### 문제 설명

<p>Steve is a hopeless perfectionist. He's been stumbling upon arrays of positive integers, but he doesn't always enjoy what he's seeing. Steve likes an array if and only if it is strictly ascending. Arrays that Steve doesn't like must be changed asap, otherwise Steve will be disappointed. The new array will satisfy Steve only if its initial length is retained, and its elements are divisible by the corresponding elements of the old array. Note that an array can be satisfactory from the start. In this case, it can be left alone (or not, if you want change).</p>

<p>Make Steve happy! Write a program that can instantly produce new, likeable arrays.</p>
