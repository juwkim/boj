### 문제 설명

<p>В одной стране все населённые пункты являются деревнями и расположены вдоль длинной прямой дороги. <p>Министерство транспорта решило запустить между деревнями рейсовые автобусы, ровно по одному маршруту <strong>из каждой деревни</strong>. Необходимо посчитать длину маршрута каждого автобуса. </p>
